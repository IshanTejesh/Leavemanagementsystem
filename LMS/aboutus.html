LEAVE MANAGEMENT SYSTEM(LMS)



About us


Introduction

This project is aimed at developing a leave management system that is of importance to any organization. The Leave Management System is an application that can be accessed by all the employees of the organization. This system can be used to automate the workflow of leave applications and their approvals.There are features like cancellation of leave,report generators etc.



Proposed Solution

Due to the inconveneince in managing the data using leave cards, it is required to have a computer based system where an employee can login and apply for the leave.He can even see the records of the previous leaves taken as well check the balance leaves.



Functional Requirements

There will be registered people in the system.some will be approvers.An approver can also be a requester.in an organization, the hierarchy will be Engineers/Managers/Managing Directors/Bussiness Managers.you can add more positions in the hierarchy.
